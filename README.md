# fixed-table
基于 Vue 2 的 table 组件库，可以上下左右四个边界冻结行列，兼容性良好，视觉体检良好。

## 安装

```bash
$ tnpm i -S fixed-table
# OR
$ npm i -S fixed-table
# OR
$ yarn add fixed-table
```

## 使用
### 完整引入

```js
import Vue from 'vue'
import FixedTable from 'fixed-table'
import 'fixed-table/dist/fixed-table.css'

Vue.use(FixedTable)
```
